Attività 3 Come sono?

Listen to the description of six Italian teenagers. Type in the adjectives in italiano, select maschile if it is describing a boy and femminile if it is describing a girl. Then write each adjective for each personality in inglese. You can listen to the audio as many times as you need to complete this activity.

Go back to Come sei? to check your understanding of adjectives.

Great work! In Activity 4 you’ll practice using adjectives to describe other people. Let’s go!
