Syntax of descriptive phrases

If you are describing something, there is a key difference in the word order (syntax) between Indonesian and English. Can you work it out?

topi merah red hat
jins biru blue jeans
teman baik good friend
anak jahat evil child

In Indonesian, the description follows what is described. The main object comes first. The description follows.

TOD rule

Where you are describing something and there is an owner in the phrase, a useful rule to remember the word order is the TOD rule. TOD stands for: Thing Owner Description
For example:

My cat is brown. Kucing saya cokelat

The main thing is the cat, the owner is ‘my’ and the description is ‘brown’.

The word ‘is’ does not translate directly into Indonesian and is not always necessary.

Where there is more than one owner present, as in the sentence ‘My mother’s hair is brown’, you may need to ask yourself some questions to get the word order correct.

What is the main thing being described? Hair
Whose hair? Mother’s
Whose mother? My
What’s it like? Brown
