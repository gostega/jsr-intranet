Tulislah!

Use the work you have done and the things you have thought about and worked through in this section to write about your own cultural context in Indonesian.

Look at how Project Pop described Indonesia, and choose a way to describe your town, suburb, school, a club you belong to or something else.

Try to answer the question: Apa yang dapat menyatukan kita?

There are a number of angles you could use to approach this (and a number of text-types; for example, you could make a web page, write an email to an Indonesian exchange student, or compose a short magazine article). Consider how you would describe one of your own communities (eg school, family group, nation, suburb, club).

You should use:

  • adjectives with correct word order for description (remember the TOD rule)
  • ada
  • link words
  • auxiliary verbs (you will find more of these in the pojok bahasa, but also review the auxiliary verbs from section three)
  • sequencing words and time indicators (see the pojok bahasa in section three).

Use the following rubric to gauge how your writing skills are developing.

 
Linguistic resources I can write short simple sentences, starting nearly all sentences in the same way. I can extend and adapt sentences to create variety and I can use simple sequencing devices and experiment a little. I can write paragraphs using time indicators and sequencing words to organise information.
Word order I use the same word order as English I modify the word order to follow simple Indonesian patterns. I put sentences together in different ways to create interest and variety.
Content I can write on familiar topics about my own personal experience. I can extend my writing beyond my immediate experience. I can write texts on less familiar topics, expressing myself with a personal style.
Support I write with guidance, or using a model. I regularly ask for help on how to find information I need. I can write original texts, using a range of sources of support (eg dictionary, notes). I can write original texts, independently selecting sources of support.
