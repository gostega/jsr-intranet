Sudah bisa? Mau nggak?

There are a number of other words which are very commonly used in both questions and statements.

You have already learned the words sudah, belum, suka and senang, which can all be used in questions as well as statements.

Three other very useful words are bisa, mau, and nggak.


  • Bisa
  • Mau
  • Nggak

Bisa can be used as a question ('Can you … ?') or a statement ('I can …'). Look at the following exchange between Brack and Suci.

Suci:    Brack bisa bahasa Indonesia?

Brack:  Bisa sedikit.

Mau can also be used the same way ('Do you want …?' or 'I want …').

Suci: Mau minum?

Brack: Tidak mau, terima kasih.

Suci: Diego, mau minum?

Diego: Boleh.

Suci: Mau minum apa?

Diego: Air saja.

Rather than responding positively with mau, Diego used the more polite response boleh, which is literally 'may' but in this context is more like 'OK', or 'yes, please'. Saying 'boleh' sounds less eager and abrupt.

Sometimes nggak (the colloquial form of 'no') is added in casual conversations. Using nggak:

  • emphasises that it is a question
  • indicates uncertainty
  • indicates that a response is needed
  • makes the question sound a little less abrupt.

Compare the following:

  • Suka makan ayam?
  • Suka makan ayam, nggak?
  • Mau pergi ke Indonesia?
  • Mau nggak pergi ke Indonesia?

Did you notice that a number of the questions used neither question words (such as apakah) nor personal pronouns (such as kamu)? For example, the formal full version of the question 'suka makan ayam?' would be 'Apakah kamu suka makan ayam?', or 'Sukakah kamu makan ayam?'. (The -kah suffix can make a non-question word a question.) However, in conversation, the question word and the word 'you' is often dropped because the tone and the context make them unnecessary.

Latihan

Choose the most appropriate response to the following questions. There may be more than one correct option.

  1. Bisa bahasa Indonesia?

    • Bisa Benar! This is correct 'I can', but there is a more modest option.
    • Bisa sedikit Benar. Pintar! This is the most correct option.
    • Boleh Salah! Boleh would indicate that you are choosing to speak Indonesian (or not), so it is not appropriate here.
    • Tidak bisa. Salah! This is obviously incorrect if you understood the question! You may be attempting humour, but sarcasm doesn’t translate well.

  2. Sudah pernah ke Indonesia?

    • Sudah. Saya sudah ke Bali dua kali. Benar! This is an appropriate response.
    • Belum Benar! This is an appropriate response.
    • Tidak. Salah! There is a better response than this if you have not yet been to Indonesia.

  3. Mau menonton film?

    • Mau This response is OK, but there is a better one.
    • Tidak mau Salah! This sounds a little rude.
    • Maaf, saya tidak bisa Benar! This is a polite and appropriate response.
    • Boleh. Benar! This is an appropriate response.

  4. Sudah punya pacar?

    • Ya Salah! There is a better response than this.
    • Sudah Benar! This is an appropriate response.
    • Belum Benar! This is an appropriate response.
    • Tidak. Salah! There is a better response than this.

  5. Mau nggak pergi ke Indonesia?

    • Mau Benar! This is an appropriate response.
    • Boleh Salah! This sounds as though you are ambivalent and don’t really care.
    • Tidak mau Salah! This sounds rather impolite and could offend.
    • Nggak mau. Salah! This sounds rather impolite and could offend.
