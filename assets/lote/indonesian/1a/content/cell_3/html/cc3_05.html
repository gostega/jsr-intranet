You have seen the words tidak and kurang used as ‘not’ or ‘not so’. Diego also uses the words:

  • jarang.

Bagiamana sifat Anda?

Bagaimana is a question word. If you hear bagaimana used, you know you are being asked: ‘What is (something) like?’ Bagaimana is really asking for a description.

Tulislah!

In Indonesian, write a paragraph about your own sifat. Save it in a Word document. You will use this later. You could:

  • use the sentence structures you practised above
  • explain your characteristics according to yourself and other people
  • say what you are not like
  • experiment with using some new words. Some really easy ones to use that work well in this context are: selalu, sering, kadang-kadang and jarang.
