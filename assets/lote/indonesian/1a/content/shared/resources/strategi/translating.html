Translating and interpreting conventions

When translating or interpreting, you should make sense in the language you are speaking or writing. So if you are translating from Indonesian to English, what you write should make sense in English. If you are interpreting into Indonesian, you should make sure you make sense in Indonesian, not translate word for word. For example, there is no word for 'is' in Indonesian. So to translate the sentence 'Teman saya lucu' into English, you would write 'my friend is funny', not 'my friend funny'.
