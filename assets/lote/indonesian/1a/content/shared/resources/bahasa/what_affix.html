What is an affix?

Affix refers to either a prefix or a suffix.

A prefix is a set of letters that go on the start of a word to modify it. For example, in English:

  Prefix Base-word
miscommunicate mis- communicate 
undo un- do
disappear dis- appear

A suffix is a set of letters that go on the end of a word to modify it.

  Suffix Base-word
communicative -ive communicate 
helpful -ful help
sadly -ly sad

A circumfix refers to when both a prefix and a suffix are applied to a word.

  Prefix Suffix Base-word
unhelpful un- -ful help    
unhappily un- -ly happy
