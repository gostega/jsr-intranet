Benar Brendon. Tapi ter- ada juga arti ‘can be’. So the following pairs of sentences have the same basic meaning:

Kota Perth dapat dilihat dari Taman Kings. Kota Perth terlihat dari Taman Kings.
Suara ribuan burung dapat didengar di taman itu. Suara ribuan burung terdengar di taman itu.
