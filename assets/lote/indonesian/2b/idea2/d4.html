Di kuliah Bahasa Indonesia hari ini, dikatakan bahwa kata sambung dan kata sambung urutan penting untuk membuat lancar tulisannya. Conjunction itu bisa mengaitkan dua paragraf supaya lebih lancar untuk dibaca. Jadi inilah beberapa kata yang efektif untuk sequencing.

Linking and sequencing

  • Pertama-tama                  Firstly

  • Yang kedua                      Secondly

  • Yang ketiga                      Thirdly

  • Akhirnya                           Finally

  • sehingga                          so that/to the point that
    Sayangnya, perusahan itu terancam bangkrut, sehingga diharuskan putus hubungan kerja kebanyakan pekerjanya.

  • walupun begitu                although that is so
    Walaupun begitu, saya mendapat pekerjaan lain dengan cepat.

  • supaya                             so that (intentional action)
    Saya belajar bahasa Inggris di Institut Bahasa pada hari Sabtu, dan ikut chatting dalam bahasa Inggris di Internet supaya dapat menjadi lebih lancar bahasa Inggris.

  • oleh karena itu                 because of that/for those reasons
    Oleh karena itu, saya mendapat pekerjaan di Happy Burger yang populer bagi turis dari luar negeri.

  • akan tetapi                       however/although that is so
    Akan tetapi, tidak ada banyak kesempatan untuk belajar bahasa Inggris.

  • namun                              however
    Namun, saya masih berusaha untuk mencari kesempatan lain diluar dari tempat pekerjaan ini.

  • meskipun                          however
    Meskipun begitu, tidak ada banyak lowongan kerja di kota saya.

  • sebaliknya                        on the other hand/conversely
    Sebaliknya, di Jakarta saya dengar ada banyak kesempatan.

  • maka                                then, so
    Saya sudah menguasai banyak tugas di posisi saya sekarang, makanya saya mencari tantangan baru.
