Re: Buku panduan, dari Ibu_Indah pada jam 08:41.

Your boss has used -lah for the verbs because they are imperatives. You’ll need to remember to remove it. You’ll also need to add a verb auxiliary (harus or seharusnya are best and interchangeable) to the sentence.

The first one has been done for you as an example.

Pelanggan seharusnya disambut dengan senyum manis.

Re: Buku panduan, dari IndoGuRu pada jam 20:21.

I think it would be better if you write the instructions in second person object focus, so that it sounds more like a direct instruction.

Rewrite the sentences below in second person object focus, using Anda rather than kamu as more suited to a formal written context. Check your answers by listening to your boss read them aloud as she corrects your new draft.

The first one has been done for you.

Example:    Kamu seharusnya menyambut pelanggan dengan senyum manis.
